## ✨ Características

- Carga y validación de archivos Excel de ovitrampas, con auto-detección de columnas.
- Tablero con indicadores clave (KPIs): ovitrampas instaladas, recuperadas, positividad y total de huevos.
- Gráficos interactivos para análisis temporal y por sector.
- Mapa interactivo de ovitrampas con agrupamiento (clusters) y mapa de calor.
- Historial de datasets guardados de forma local.
- Gestión de usuarios y roles (admin / analista / visor).
