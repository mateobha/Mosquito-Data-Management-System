# Mosquito Data Management System v1.0.1

Sistema de escritorio para el monitoreo de ovitrampas

## 📦 Requisitos
- Windows 10 o superior (64 bits)
- ~1 GB de espacio en disco
- Permisos de administrador para instalar

## ⬇️ Instalación
1. Descarga `MosquitoDataManagementSystem_Setup_v1.0.1.exe` desde los *Assets* de esta release.
2. Verifica el hash SHA-256 (ver abajo).
3. Ejecuta el instalador y sigue el asistente.

Guía detallada: [docs/instalacion.md](docs/instalacion.md)

## 🗂️ Archivo
- **Nombre:** `MosquitoDataManagementSystem_Setup_v1.0.1.exe`

## 🔐 Hash SHA-256
```
DC3A802F7C4939BE03736C92719454C920B4514BDCE7B9EFAEABB24A4242FBF2
```
Verifícalo con:
```powershell
Get-FileHash -Algorithm SHA256 "MosquitoDataManagementSystem_Setup_v1.0.1.exe" | Format-List
```
